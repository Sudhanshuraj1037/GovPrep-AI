## 📖 Overview

GovPrep AI is a full-stack web application designed to help students prepare for competitive government examinations. It uses AI to generate customized practice tests, provides an AI-powered study assistant, supports PDF uploads, and offers a modern dashboard for managing learning progress.

The project is built using React for the frontend and Node.js + Express for the backend.

- 🔐 Secure Login & Signup

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Sudhanshuraj1037/GovPrep-AI.git
```

Move into the project directory:

```bash
cd GovPrep-AI
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
JWT_SECRET=YOUR_SECRET_KEY
PORT=5000
```
